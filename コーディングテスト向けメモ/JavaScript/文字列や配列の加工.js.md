### [Array.join()](https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Global_Objects/Array/join) : 前配列要素を任意の文字列を挟みながら連結する。
### [String.split()](https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Global_Objects/String/split) : 文字列を任意の文字を敷居として配列要素に分割する。

コード例  
~~~js
A = [1,2,3,4,5]
B = A.join("-");
console.log(B);
C = B.split("3");
console.log(C);
~~~
出力結果  
~~~
1-2-3-4-5
[ '1-2-', '-4-5' ]
~~~
